# Pokemon Combat Simulator

An interactive Pokemon battle simulator built with Streamlit, using real data from the PokeAPI.

## Deployed App

[Click here to open the app](https://group5-pokemon-combat-simulator.streamlit.app/)

## Features

- Select any two Pokemon from the original 151
- View sprites, types, and base stats
- Choose damaging moves for each Pokemon
- Compare stats with a grouped bar chart
- Simulate turn-based battles with real Pokemon damage formulas
- View battle logs and HP over time charts
